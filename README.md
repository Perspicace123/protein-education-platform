# 3D Protein Structure Interactive Learning Platform

Platform Introduction
A 3D protein interactive learning platform for middle school biology education. Here, students can directly explore the three-dimensional structure of proteins through intuitive operations, observe the interactions between molecules, and make abstract biological concepts vivid and visible.
